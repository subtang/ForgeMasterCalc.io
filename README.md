# Forge-Master-calculator
## 대장간 키우기 계산기

# 1. 대장간 키우기 계산기
## 1-1. 본인의 현재 장비 스펙과 교체할 장비 스펙을 입력하거나 혹은 이미지를 스캔하여서 장비 스펙을 자동으로 입력받아 계산합니다.

# 2. 아직 개발중인 버전입니다.
## 2-1. 이미지 인식이 완벽하지 않습니다. 인식률은 80% 정도로 꽤나 높습니다만 ' . ' 에 대한 부분이나 수치를 잘못인식하여 제대로 입력되지 않을 수 있습니다
## 2-2. 혹시 모르니 비교하셔서 잘못된 부분을 수정하시면 됩니다.
## 2-3. 피드백 및 수정사항 , 건의사항 받고있습니다. 검토 후 추후 더욱 기능을 늘려가겠습니다.
